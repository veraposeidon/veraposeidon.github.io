ProcessOn和WWWSQLDesigner

发表于 2018-02-02 更新于 2023-04-02 分类于 工具分享
简单介绍：在线作图工具ProcessOn 和 在线数据库可视化设计工具 WWW SQL Designer

今天推荐两款在线工具，这些都是一个浏览器就能发挥的工具：

  • ProcessOn：在线流程图、原型图等作图工具；
  • WWW SQL Designer：在线数据库表设计工具。

1. ProcessOn

官网链接：https://www.processon.com/

登录方式：微信、QQ、微博、Google

优点：

  • 在线作图工具的聚合平台，它可以在线画流程图、思维导图、UI原型图、UML、网络拓扑图、组织结构图
  • 可以把作品分享给团队成员或好友，无论何时何地大家都可以对作品进行编辑、阅读和评论
  • 拥有海量模板（其实模板并不多，比没有好）

缺点：

  • 免费用户只有9个文件数量上限

同类型产品：百度脑图， http://naotu.baidu.com/ 也很棒，就是登录百度账号比较麻烦。

2. WWW SQL Designer

WWW SQL Designer 是一个开源项目，提供了在线的数据库表设计功能，用户无需注册或者登陆即可直接使用。

项目地址：https://github.com/ondras/wwwsqldesigner

Demo地址：http://ondras.zarovi.cz/sql/demo/?keyword=default

用户手册：https://github.com/ondras/wwwsqldesigner/wiki/Manual

支持的数据库功能：

  • 键
  • 强制外关键字
  • 注解
  • 索引

创建完数据库图像后，你可以：

  • 保存和加载自己的数据库表设计
  • 打印
  • 作为SQL脚本导出

使用介绍

同类型产品：Pony ORM，https://editor.ponyorm.com/ 没用过，看起来很强大，就是需要登录比较麻烦，还有需要注意的一点就是免费用户的设计貌似属于public，别人可以看到。
